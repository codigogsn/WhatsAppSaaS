Nueva Regla de Upsell

Cuando un cliente ordena de una categoria, el bot puede sugerir un producto adicional para aumentar el ticket (ej: "hamburguesa" → sugerir "bebida").
Vincula un producto del menu o usa texto libre como alternativa.

Importar Menu

Pega texto estructurado o JSON para importar categorias, items, precios y aliases de una sola vez.
Tambien puedes incluir secciones Extras: y Upsells: para importar configuracion adicional.
Tip: Usa Claude o ChatGPT para convertir un PDF de menu a texto estructurado, luego pegalo aqui.
